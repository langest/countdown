URL format details

This page reads repeated event parameters using: event=YYYY-MM-DD|HH:MM|TimeZone|Title

Time zone can be an IANA name (recommended) such as America/New_York or a UTC offset such as -05:00.

Example:

?event=2026-12-31|23:59|America/New_York|New%20Year&event=2026-07-04|09:00|Europe/Paris|Parade
