# Workspace Agent Notes

## Purpose
Static GitHub Pages project for URL-driven countdown cards.

## Navigation
- `index.html`: Main page containing form mode and countdown mode.
- `styles.css`: Visual styling and responsive layout.
- `app.js`: URL parsing, rendering, sorting, and countdown ticking.
- `PLAN.md`: Active plan, findings, and progress notes.

## Working Conventions
- Keep implementation simple, readable, and secure.
- Prefer small focused functions and early returns.
- Keep docs updated as behavior changes.
